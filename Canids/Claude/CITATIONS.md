## Example Citation in Methods Section

> "Orthologous sequences were aligned using MAFFT v7.490 (Katoh & Standley, 2013) and back-translated to codon alignments using Pal2Nal (Suyama et al., 2006). Selection analyses were performed using HyPhy 2.5 (Kosakovsky Pond et al., 2020). We applied aBSREL (Smith et al., 2015) to detect episodic positive selection, BUSTED (Murrell et al., 2015) to identify genes under selection in foreground lineages, and RELAX (Wertheim et al., 2015) to test for intensification or relaxation of selection. Functional enrichment analysis was conducted using gprofiler2 (Kolberg et al., 2020). The entire workflow was automated using Snakemake (Mölder et al., 2021)."
